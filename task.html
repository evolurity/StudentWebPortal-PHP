Студенческий портал

Постановка задачи:

Цели проекта:

Создать Веб-Блог, который позволяет размещать учебные статьи, материалы для студентов. С возможностью авторизации для администратора, подключенной БД, возможностью комментировать записи, фиксировать количество просмотров, выводить топ читаемых статей, постраничной навигацией, сортировкой по количеству просмотров, вывод новейших комментариев и записей, сортировкой записей по категориям и пр..

Ожидаемые результаты:

Разработанное и размещенное на веб-хостинге sprinthost клиент-серверное веб-приложение, которое будет выполнять функции студенческого Веб-сервиса, выступающего в роли полезного ресурса с учебными статьями и материалами для студентов. Данное приложение способно выступать так же в роли информационного портала для уведомления студентов о возможных стажировках/практиках и последних новостях из мира информационных технологий.

Задачи, которые необходимо решить:

Создание базы данных и логики администрирования. Создание веб-интерфейса, позволяющего пользователю просматривать, комментировать статьи и записи,производить навигацию по страницам, категориям, просматривать новейшие записи и комментарии, а администраторам заходить в панель администрирования и модерировать любую информацию и содержание Веб-сайта, создавать новые записи, удалять или редактировать существующие.

Методы решения задач:

Проектирование Веб-Блогов: авторизация, логика взаимодействия с Базой Данных MySql, phpmyadmin, создание удобного интерфейса для клиента.

Инструментальные средства, которые предполагается использовать для разработки приложения:

Основная логика приложения будет выполнена на языке PHP, а разметка веб-страниц с использованием HTML и CSS.Запросы в базу данных MySQL

Описание планируемого функционала приложения и интерфейса пользователя:

Для пользователя: возможность просмотра, оставления коммментариев к записям и другим учебным материалам. Реализация в виде полноценного удобного интерфейса Веб-Портала.

Для администратора: возможность модерировать сайт, управление собранными данными, записями и содержимым сайта.
